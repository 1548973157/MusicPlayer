# QT 音乐播放器

一个简单的本地音乐播放器，支持基本播放控制。

## 功能
- 播放/暂停/继续
- 音量调节
- 文件选择
- 显示当前歌曲名

## 使用方法
1. 点击"选择音乐"按钮选文件
2. 使用播放/暂停按钮控制
3. 拖动滑块调音量
